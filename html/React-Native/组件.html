text组件 #

  • text组件主要是用来显示文本,具有响应性,可以嵌套,可以继承样式
  • API:
    • onPress: 文本被点击的时候出发的回调函数
    • numberOfLines: 用于设置显示的最大文本数量，超过部分会用省略号代替
    • allowFontScaling: IOS特有，用于控制字体是否要根据iOS的“文本大小”辅助选项来进行缩放
