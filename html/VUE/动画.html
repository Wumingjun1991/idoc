动画 #

引入动画库 #

需要用link标签引入CSS动画库,调用其中不同的动画名来实现不同的动画效果,1.0和2.0版本中有一定区别

1.0版本 # 

<div id="app">
    <button @click="flag=!flag">点击显示或者隐藏</button>
    <div v-show="flag" class="fade animated" transition="fade"></div>
    <!--
    引入动画库:
    1.给当前执行动画的元素加上class名=animated
    2.给当前vue实例加一个属性transitions{}
    3.给当前这个transitions属性中增加一个属性(这个属性和标签上transition的值一样),比如这里写的fade
    4.给fade(动画名)增加两个属性
    enterClass:'fadeInUp'
    leaveClass:'fadeOutDown'
    -->
</div>
<script src="js/vue1.0.js"></script>
<script>

    var vm = new Vue({
        el:'#app',
        data:{
            flag:true
        },
        transitions:{
            fade:{
                enterClass:'fadeInUp',
                leaveClass:'fadeOutDown',
            }
        }
    })
</script>

2.0版本 # 

<div id="app">
    <button @click="flag=!flag">点击显示或者隐藏</button>
    <br>
    <!--如果是多个元素要实现动画,就要使用transition-group标签-->
    <transition-group enter-active-class="animated fadeInUp" leave-active-class="animated fadeOutDown">
        <div v-show="flag" class="fade animated" :key="1"></div>
        <div v-show="flag" class="fade animated" :key="2"></div>
        <div v-show="flag" class="fade animated" :key="3"></div>
    </transition-group>
</div>
<script src="js/vue2.0.js"></script>
<script>

    var vm = new Vue({
        el:'#app',
        data:{
            flag:true
        },
        transitions:{
            fade:{
                enterClass:'fadeInUp',
                leaveClass:'fadeOutDown',
            }
        }
    })
</script>

自定义动画 #

Vue中除了引入动画库之外,还可以自定义动画,1.0和2.0版本中也是有不同的设置方式 Vue的自定义动画主要是通过本身来控制CSS样式的改变来实现的

1.0版本 # 

  <style>
        .fade{
            margin-top: 10px;
            width: 100px;
            height: 100px;
            background-color: aqua;
        }
        .fade-transition{
            transition: 1s linear;
            -webkit-transition: 1s linear;
        }
        .fade-enter{
            /*进入的样式*/
            /*opacity: 0; 想让透明度从0变到1*/
            opacity: 0;
            /*height:0; 想让高度从0变到100px*/
            height: 0;
        }
        .fade-leave{
            /*离开的样式*/
            /*opacity:0; 想让透明度从1变到0*/
            opacity: 0;
            height: 0;
        }
    </style>
<div id="app">
    <button @click="flag=!flag">点击显示或者隐藏</button>
    <div v-show="flag" class="fade" transition="fade"></div>
</div>
<script src="js/vue1.0.js"></script>
<script>
    var vm = new Vue({
        el:'#app',
        data:{
            flag:true
        }
    })
</script>

2.0版本 # 


 <style>
        .fade{
            margin-top: 10px;
            width: 100px;
            height: 100px;
            background-color: aqua;
        }
        /*fade-enter         进入的状态
          fade-enter-active  触发进入状态,这里只需要写进入时的transition
          fade-leave         离开的状态
          fade-leave-active  触发离开状态,这里只需要写离开时的transition
        */
        .fade-enter{
            width: 50px;
            height:50px;
        }
        .fade-enter-active{
            transition: 1s linear;
        }
        .fade-leave-active{
            width:0;
            height:0;
            transition: 1s;
        }
    </style>
<div id="app">
    <button @click="flag=!flag">点击显示或者隐藏</button>
    <transition name="fade">
        <div v-show="flag" class="fade"></div>
    </transition>
</div>
<script src="js/vue2.0.js"></script>
<script>
    var vm = new Vue({
        el:'#app',
        data:{
            flag:true
        }
    })
</script>
